Καλώς ήλθατε σε μια ιστοσελίδα παροχής πληροφοριών για τα μουσεία της Αθήνας!

Είσαι μαθητής/τρια ή καθηγητής/τρια πρωτοβάθμιας ή δευτεροβάθμιας εκπαίδευσης; Αναζητάς συγκεντρωμένες όλες τις πληροφορίες για τα μουσεία της Αθήνας; Θέλεις να οργανώσεις ή να προτείνεις μια επίσκεψη-εκπαιδευτική εκδρομή σε ένα ή περισσότερα από αυτά;

Αν ναι τότε αυτή η ιστοσελίδα είναι για εσένα!

H ιστοσελίδα αυτή, σου δίνει τη δυνατότητα είτε να περιηγηθείς σε όλα τα μουσεία και να βρεις όλες τις πληροφορίες για αυτά, είτε να επιλέξεις συγκεκριμένα μουσεία και να τα συγκρίνεις.

Για να ξεκινήσεις να βλέπεις τα μουσεία, χρησιμοποίησε την καρτέλα πλοήγησης που φαίνεται στο πάνω μέρος της σελίδας.
